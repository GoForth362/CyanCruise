# CyanCruise 就业岗位知识库

## 就业方向

CyanCruise 重点支持计算机、软件工程、信息管理等专业学生的就业规划。常见方向包括 Java 后端开发、前端开发、测试开发、数据分析、产品助理、运维开发、实施顾问和企业数字化顾问。

## Java 后端岗位要求

- 编程基础：熟悉 Java 语法、集合、异常、泛型、并发基础和 JVM 基本概念。
- Web 开发：理解 HTTP、REST 接口、认证授权、参数校验、异常处理和日志。
- 数据库：掌握 SQL、索引、事务、表设计和常见性能问题。
- 框架能力：了解 Spring Boot、MyBatis、缓存、消息队列等常见企业开发技术。
- 工程能力：会使用 Git、Maven 或 Gradle，理解分层架构和接口契约。
- 项目表达：能够讲清业务背景、个人职责、技术方案、难点、结果和复盘。

## 城市与准备策略

杭州、深圳、广州、上海、北京等城市互联网和企业软件机会较多。若学生缺少实习，应优先补强项目质量、简历表达、基础题训练和面试讲述。若学生已有项目但技术深度不足，应增加接口设计、数据建模、权限控制、性能优化或部署运维等亮点。

## 行动计划模板

- 第 1 阶段：补齐 Java、数据库、计算机网络和操作系统基础。
- 第 2 阶段：完善 1 到 2 个可讲述项目，形成接口文档、数据库设计和部署说明。
- 第 3 阶段：完成简历初稿，按岗位要求调整项目描述。
- 第 4 阶段：进行笔试题、八股题、项目面试和行为面试训练。
- 第 5 阶段：按城市和岗位梯度投递，复盘面试反馈。

## 输出要求

建议应使用普通学生能理解的中文，明确下一步行动、时间安排和优先级。不要只堆技术名词。
