# CyanCruise 升学规划知识库

## 考研规划总览

CyanCruise 面向计算机、软件工程等专业学生提供升学规划建议。规划时应结合学生年级、专业基础、成绩排名、数学和英语水平、目标城市、目标院校层次、家庭支持情况、是否接受调剂、是否有保研资格等信息。

## 时间线

- 大一至大二：夯实高等数学、线性代数、概率论、英语、专业核心课，参与课程项目和科研训练。
- 大三上：确认升学意向，评估保研或考研可能性，初步筛选院校专业。
- 大三下：确定目标院校和考试科目，完成第一轮基础复习。
- 暑假：强化数学、专业课和英语阅读，开始政治基础学习。
- 大四上：完成真题训练、模拟考试、错题复盘和报名确认。
- 初试后：准备复试材料、项目讲述、专业面试和英语口语。

## 决策维度

- 成绩与排名：排名靠前且科研或竞赛基础较好，可优先评估保研；成绩稳定但排名一般，可考虑考研。
- 专业基础：数学和专业课基础强，适合冲刺更高层次院校；基础一般时应选择稳妥梯度。
- 职业目标：希望进入科研、算法、学校、国企或需要学历门槛岗位时，升学收益更高。
- 实践经历：实习和项目较强时，就业也具备竞争力；实践薄弱但学习能力强时，可通过升学补足竞争力。
- 风险承受：考研存在不确定性，应准备就业备选方案。

## 输出要求

建议应包含目标判断、信息缺口、院校或方向梯度、每月行动计划、风险提示和备选路径。避免只给结论，应说明理由。
